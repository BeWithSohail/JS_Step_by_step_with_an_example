In JavaScript, strings are used to represent and work with a sequence of characters. A string can represent an object as well as the primitive data type. JavaScript automatically converts primitive strings to String objects so that it's possible to use String methods and access properties even for primitive strings. In this reference page, you will find all String methods and properties available in JavaScript. For example, the toUpperCase() method returns the string converted to uppercase.
